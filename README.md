## Deskripsi
Program ini menunjukkan bagaimana Python dapat diintegrasikan untuk melakukan automasi backup folder secara sederhana.

## Persiapan
1. Pastikan Python 3.8+ sudah terinstall
2. Download file `backup_automation.py`
3. Simpan di folder yang diinginkan

```bash

# 2. Jalankan program
